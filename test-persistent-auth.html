Technical Details

Storage Keys:
- browserPageImpact_token: Stores authentication token with expiration
- browserPageImpact_docId: Stores connected Google Doc ID

Token Structure:
{
  "access_token": "ya29.a0...",
  "expiresAt": 1625097600000  // Unix timestamp
}

Expiration Handling:
- Tokens are checked for expiration with 5-minute buffer
- Expired tokens are automatically removed from storage
- API calls with expired tokens trigger re-authentication
