🎯 How It Works

🔐 Secure Authentication

Uses Google OAuth for secure access to your documents

📅 Monthly Organization

Automatically organizes entries by month with proper formatting

🚀 One-Click Access

Works on any website - just click the bookmark!

💾 Remembers Settings

Saves your document ID and preferences locally

📝 Usage

  1. Click the bookmarklet on any webpage
  2. Select the month you want to add content to
  3. Enter a title for your entry
  4. Write your content in the text area
  5. Click "Add Entry" to save to your Google Doc

✨ Features

  • Auto-creates month sections if they don't exist
  • Timestamps each entry automatically
  • Works on any website - no hosting required
  • Remembers your document between sessions
  • Clean, modern interface that won't interfere with websites

🔧 Troubleshooting

Authentication Issues

Make sure you've added the correct authorized JavaScript origins in your Google Cloud Console for the domains where you'll use the bookmarklet.

Document Access Issues

Ensure you have edit access to the Google Doc and that the document ID is correct.

Bookmarklet Not Working

Try refreshing the page and clicking the bookmarklet again. Some websites may block certain scripts.
